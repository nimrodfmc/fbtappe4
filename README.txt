Cette application est une application minceur dans laquelle l'utilisateur doi rentrer des données le concernant (il doit saisir des
informations tels que le nom ou le prenom, mais aussi la nourriture qu'il mange à chaque repas), afin de déterminer combien de calories
il mange par jour, afin de pouvoir "limiter" sa consommation et donc rentrer dans la vocation de l'application, à savoir mincir.
